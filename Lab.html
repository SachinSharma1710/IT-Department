Department of Information and Technology

Lab

Department of Information Technology has well-equipped laboratories where students can implement their programs and hone their programming skills.

  • Programming Laboratory

    This laboratory is solely configured for improving the programming skills of students. The laboratory contains necessary software and frameworks like C, C++, Core JAVA, Advance JAVA and Android Studio.

  • Web Technology Laboratory

    This laboratory has systems with the latest configurations with which students can perform experiments related to different web development technologies like HTML 5, Angular JS, Node JS, Python and CSS.

  • Computer Concepts Laboratory

    Computer concepts laboratory has PCs with the power of i-3 or core-2 duo high-speed processors and is well equipped with various open-source and licensed software to learn Computer Graphics, DBMS, OOPS, Data Structure, ADA and Operating Systems like Linux and Windows.

  • Advance Computing Laboratory

    This laboratory comprises of advance i-3 8th generation computer systems to demonstrate and learn the concepts of latest technologies like Cloud Computing, Machine Learning, Artificial Intelligence and Internet of Things (IOT).

  • Project Laboratory

    Project laboratory provides an environment where students can develop minor and major projects with their team members. This laboratory is equipped with all kinds of software which are required to implement junior and senior projects.
